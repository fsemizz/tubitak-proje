# Akıllı Tahta Kodlama (MVP)

Bu sürüm, okul öncesi ve ilkokul için akıllı tahta uyumlu, web tabanlı (PWA) bir etkinlik prototipidir.

## GitHub Pages
- Bu repo kökünden yayınlanacak şekilde hazırdır.
- `Settings → Pages → Deploy from branch` ile `main` ve `/root` seçin.

## QR / Etkinlik Linki
- Linke `?e=OKN-01` gibi bir parametre ekleyin.
- Örnek: `https://fsemizz.github.io/tubitak-proje/?e=OKN-01`

## Geliştirme Notları
- Kartlar dokunarak eklenir.
- PWA offline cache içerir.
